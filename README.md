# Site

Site for 12urenloop Gent.

## Dev

A static site build with [nanoc](https://nanoc.ws/).

Things you might wanna do:

- install rbenv / ruby
- `gem install bundler`
- `bundle install`

Then run the code with `bundle exec nanoc` or `bundle exec nanoc live`.

Deploy with `bundle exec nanoc deploy`.
